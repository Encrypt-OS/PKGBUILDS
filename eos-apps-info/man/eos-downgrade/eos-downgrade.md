# eos-downgrade

Downgrade EndeavourOS (and Arch) packages.

Note: Certain EndeavourOS files will be fetched from github.com.<br>This may cause issues in some locations.

## Synopsis
```
eos-downgrade [options] <package-name(s)>
Options:
    -h, --help     This help.
```

## Example
```
eos-downgrade akm
```
